### 🤖 AI Document Chatbot (RAG)

A full-stack AI chatbot that allows users to upload PDFs, DOCX files, or website URLs and ask questions about their content. The system retrieves relevant information and generates accurate answers with source references.

**Highlights:**
- Retrieval-Augmented Generation (RAG) architecture  
- FAISS vector search + embeddings
